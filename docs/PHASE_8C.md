# Phase 8C：移动端访问与本机后台限制

本阶段处理局域网移动端访问和后台账号安全边界。

完成内容：

- 前端 API 默认地址按当前访问主机推导为 `协议://当前主机:8080/api`。
- 兼容既有 `/api/...` 调用，避免出现重复 `/api/api`。
- 768px 以下增加移动端布局：导航折行、首页卡片双列、详情单列、选座横向可滚动。
- 后台账号仅允许通过服务器本机 `localhost` 或 `127.0.0.1` 登录。
- `/api/admin/**` 与 `/api/checker/**` 在非本机 Host 下拒绝访问。

验证：

- `mvn -q -DskipTests package`
- `npm run build`
